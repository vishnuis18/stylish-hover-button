# Stylish Hover Button

A modern and visually appealing hover button created using HTML and CSS. The button features a glowing neon effect and smooth hover animations, making it suitable for modern web interfaces and landing pages.

## Features

- Neon glow effect
- Smooth hover animation
- Pure HTML and CSS
- Responsive design
- Modern UI appearance

## Technologies Used

- HTML5
- CSS3
